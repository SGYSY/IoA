IoA Installation#


Manual Installation ( Recommended!! )#

Tip

Ensure your Python version >= 3.10 (It is advisable to use version 3.10 universally).

  1. Use conda in the command line to create a python environment.

    conda create --name your_env_name python==3.10

  1. Clone IoA to your local machine.

    git clone https://github.com/AgentVerse/IoA.git
cd IoA

  1. Make sure you have Docker installed on your local environment. If Docker is not installed, for download details, please visit the Docker official website .


Server#

  1. Start the Milvus to ensure the server can connect to it by running. 
    docker-compose -f dockerfiles/compose/milvus.yml up --build

  1. Build the Docker image required to start the server.

    docker build -f dockerfiles/server.Dockerfile -t server:latest .

  1. Create and complete the docker-compose file your_server.yml, and start the server using docker-compose.

    docker-compose -f dockerfiles/compose/your_server.yml up

Client#

  1. Build the images required for the Agents.

    • Agent

      docker build -f dockerfiles/agent.Dockerfile -t agent:latest .

    • Tool Agent (e.g React Agent)

      docker build -f dockerfiles/tool_agents/react.Dockerfile -t react:latest .

  1. Create and complete the docker-compose file test_agent.yml, and start the Agents using docker-compose.

    docker-compose -f dockerfiles/compose/test_agent.yml up
