########################
部署IoA服务
########################

|


1. 首先进入服务器

   .. code-block:: bash

     ssh username@ip_address

|

2. 创建一个文件夹，将IoA clone到文件中，

   .. code-block:: bash
      
     cd /data 
     mkdir your_file_name
     cd /data/your_file_name
     git clone https://github.com/AgentVerse/IoA.git

|

3. 拉取IOT case中IOT Server所需要用到的image，

   .. code-block:: bash

     docker pull mlmz/iot:latest

|

4. 在 :code:`dockerfiles/compose` 目录下的 :code:`server.yml` 包含了IoA Server和IOT Server。通过docker-compose命令在服务端启动。

   .. code-block:: bash

     docker-compose -f dockerfiles/composeserver.yml up --build

|

.. tip::

   如果需要想要部署其他server，可自行修改server.yml。
